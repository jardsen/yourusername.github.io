# CHASER OS

> Personal operating system and app launcher. One place to access every tool I build.

---

## What is this?

Chaser OS is my personal hub — a password-protected home screen that lives at `yourusername.github.io` and links out to every project I build. Instead of hunting through bookmarks or repos, everything is one click away.

## How it works

- Each project lives in its own repo with GitHub Pages enabled
- Chaser OS links to each project's live URL and loads it in an iframe
- When a new project goes live, the hub gets updated with the real URL

---

## Stack

- Pure HTML / CSS / JS
- Hosted on GitHub Pages
- No frameworks, no dependencies
